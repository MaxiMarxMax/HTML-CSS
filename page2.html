Apprendre à utiliser Html/CSS

HTML

Comme dit précédement HTML c'est à la structure d'une page web, pour cela nous allons avoir besoin de ce qui s'appelle une balise.

Les Balises :

  • Balise Ouvrante : <..>
  • Balise Fermante : </..>
  • Balise Autofermante : <../>

Les balises ouvrantes et fermantes marchent toujours de paires. C'est a dire que <..> </..> est le fonctionnement normal de toutes balise.
Comparés à la balise autofermante <../> qui fonctionne toute seule.
Exemple :

  • <p></p> est une balise pour encadrer le texte dans une paragraphe, c'est a dire que votre éditeur de code va considérer le texte dans cette balise <p></p> comme un paragraphe.
  • <br/> est une balise autofermante c'est à dire qu'elle est autonome qui va servir à revenir a la ligne.

Les Attributs :

Les attributs sont des valeurs supplémentaires qui configurent ou ajustent les éléments. Les attributs sont composés de propriétés et des valeurs. Ceci se composent comme ceci :

propriété="valeur".

Le Principe D'Indentation :

Ce principe fonctionne comme les poupées russes. C'est à dire que chaque élément contenu dans les balises doit précédé de d'une préssion sur la touche :

TAB.
