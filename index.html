Les Langages

HTML

HTML veut dire HyperText Markup Language, il est utilisé pour la structure d'une page web.


Ce langage est toujours accompagné de CSS dont on parleras plus tard, afin d'avoir une maitrise dans la conception de site web, il est important de pas aller trop vite et de savoir recopier ses cours une pairs de fois. C'est en exercant qu'on devient habile et habitué à HTML.


Je suis actuellement dans l'apprentissage de ce langage, langage qui me permet d'ailleurs de faire le site sur lequel vous pouvez lire ces phrases.

CSS

CSS veut dire Cascade Style Sheets.C'est un langage basé sur un document HTML, totalement dépendant de celui car CSS va apporter les touche de couleur et de design pour HTML. HTML est a la structure d'une maison ce que CSS est a l'amenagement de la proriété.


Une multitude de tutoriel sur Google ou Youtube vont vous permettre de vous faire la main avec les langage basic d'un développeur web et web mobile.

Il y a plusieurs manieres de s'excercer, ici je vais parler d'éditeur de text :


Sublime Text

Points Forts

  • - Facile a prendre en main.
    - Intuitif et reglable plutot facilement.

Points Faibles

  • - Le rendu n'est pas instantanné, il faut sauvegarder et ouvrir le site afin de voir les modifications apporté.
    - Avec "TAB", Sublime Text va finir votre balise selon les éléments au dessus. Malheureusment ce n'est pas forcément très bien fait.
Télécharger Sublime Text

Code Pen

Points Forts

  • - Le rendu est instantanné, contrairement a Sublime Text. ( Réel Point Fort.)
    - Facile a prendre en main

Points Faibles

  • - Il n'y a pas d'indication si la ligne de code est éxact ou non.
    - Besoin d'internet pour l'utiliser.
Utiliser CodePen
