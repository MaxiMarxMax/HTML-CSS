Le Glossaire

Ici Vous êtes dans le glossaire, c'est une page web qui va constituer un lexique expliquant les mots difficles ou mal connus. Il va y être référencé les mots de vocabulaires spécialisé qui y est utilisé.

Le Header

Le Header désigne la zone situé en haut d'une page web. Il y comporte de nombreuses informations, comme le nom l'entreprise, le logo, un menu et des onglets. Il permet de donner l'identité visuelle de botre site web.

Le Footer

Le footer est la zone en bas d'une page web. Il doit comporter un certains nombre d'élément indispensable afin de permettre au utilisateur d'acceder au mentions légales, mais aussi les conditions général de ventes. Mais aussi les différents contacts possible, que ce soit les informations de contact, l'identification de l'entreprise ou le référencement naturel de la page, le footer est super important !.

La Sidebar

La sidebar (ou barre latéral) correspond à une colonne situé sur le coté de la page web. Il peut être utiliser afin d'y incorporer un menu déroulant, un formulaire d'inscription ou un champ de recherche. Cet élément est totalement personnalisable en fonction de vos besoins.

La Home Page

La home page (ou page d'acceuil) est la page principale d'un site internet. Cet page contient les principales informations de votre site web. Elle est souvent la première à laquelle le visiteur accède. C'est en quelque sorte la vitrine de votre entreprise sur le web. Selon la règle des 3 clics, le visiteur doit pouvoir effectuer une conversion en seulement 3 liens depuis la home page.

La Landing Page

Les landing pages (ou pages d'attérisage) sont les pages par lesquelles les internautes arrivent sur votre site apres avoir cliqué dessus. Les landing pages sont crées spécialement pour vos actions marketing ponctuelles. Leurs but est d'amener le visiteur vers l'achat au plus vite.

Call-To-Action

Le call-to-action est un élément INDISPENSABLE qui doit être au coeur de la réfléxion lors de la création d'un site web. Ce terme désigne des boutons créés pour inciter le visiteur à clique et menant à la conversion. Les call-to-action sont le déclencheur du processus d'achat. De ce fait, ils doivent être mis en valeur et facilement accessibles.

Le SEO

Le SEO englobe différentes techniques permettant à un site web d’être bien référencé sur Google en fonction de mots clés pertinents. Le référencement naturel doit être pris en compte dés la création d’un site web car il détermine les principaux mots clés du site, l’arborescence, le type de contenu. Le SEO fait appel à de nombreuses connaissances et est un élément indispensable à la visibilité de votre entreprise en ligne. C’est pourquoi nous vous conseillons de faire appel à des professionnels du web pur la création d’un site web.

L'Arborescence

L’arborescence est le squelette, la structure de votre site web. Ce terme désigne la manière dont les pages sont reliées entre elles pour faciliter la navigation sur votre site. L’arborescence est essentielle pour l’ergonomie de votre site d’entreprise mais elle définit également votre stratégie de référencement naturel. Il s’agit de l’une des premières étapes dans le processus de création d’un site web.

Le Back Office

Le back office est la partie immergée de l’iceberg, tous les éléments de votre site web qui ne sont pas accessibles aux visiteurs. Il s’agit des coulisses de votre site, l’interface permettant la mise en ligne de contenus et certaines modifications du site. Lors de la création du site web, vous devez vous assurer que vos équipes marketing peuvent accéder au back office afin de modifier les informations présentes sur votre site.

La PréProd

La phase de préproduction ou « préprod » est l’étape de développement précédant la mise en ligne de votre site web. La préproduction permet de construire modifier et tester les fonctionnalités du site sans que cela ait une influence sur le site en ligne. Elle est essentielle dans la création d’un site web afin que les développeurs puissent livrer un site adapté, avec le moins de bugs possibles lors de la mise en ligne.
