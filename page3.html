La Structure De Vos Lignes De Code :

  • Le Doctype :

    • Le DOCTYPE comme son nom l'indique est nécéssaire afin d'indiquer quel sera le type du document. Ici on veut que ce soit une doctype en HTML donc ce sera écrit comme ceci :

    <!DOCTYPE html>

  • Les Balises html

    • Les balises html sont celles qui vont encadrer toutes les lignes de codes. Elles sont écrits comme ceci :

    <html>...</html>

  • Les Balises Head et Body

    • Les balises Head et Body sont les deux grandes sections qui vont composer toutes vos lignes de codes. COmmes leurs traductions l'indiquent elles veulent respéctivement dire, Tête et Corps, ce qui donne déjâ un peu plus de précisions sur leurs utilités. Head est le conteneur qui va obligatoirement contenir un certains nombre de ligne de code éssentielle au bon traitement de celle-ci. Comme par exemple le type de font utilisé ou la langue dans laquelle tout cela va être compris. Cette partie la est invisible a l'utilisateur, c'est a dire quelle n'apportes rien a la mise en page.

    Par conséquent, Corps va contenir tous ce qui va etre affichés a l'utilisateur, ainsi tout ce que vous avez plus lire et voir jusqu'a présent est contenue dans la balise Body. Vous dites magie, moi je dis programation !

Voici comment est structuré une page html ( même si nous verrons plus tard que nous allons en rajouter un peu afin d'améliorer la mise en page).
